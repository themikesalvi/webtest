Phase 1: Pre-Edit Asset Preparation
[ ] Source File Received: Confirm the final, full-length podcast video file has been downloaded and saved in the correct project folder.

[ ] Opus.pro Analysis Complete: The full episode has been uploaded to Opus.pro for initial clipping and scoring.

[ ] Clips Selected: 6-10 potential clips have been identified and marked for export in Opus.pro based on their Viral Score and content relevance.

Phase 2: Initial Edit & Template Application
[ ] Clip Downloaded: The selected clip has been downloaded from Opus.pro.

[ ] Correct Template Opened: The correct client's DaVinci Resolve project template has been opened.

[ ] Clip Imported: The downloaded clip has been imported into the DaVinci Resolve Media Pool for this project.

[ ] Clip Dropped on Timeline: The clip has been placed onto the appropriate video track in the template timeline.

Phase 3: Content Refinement & Timing
[ ] Hook Optimized: The first 3-5 seconds have been reviewed. The in-point has been trimmed if necessary to ensure the most impactful statement is at the very beginning.

[ ] Pacing Checked: The clip's pacing feels natural. Unnecessary pauses, "ums," or "ahs" have been trimmed.

[ ] Duration Finalized: The clip length is within the target range (15-90 seconds).






Phase 4: Technical QA
[ ] Video Sync Verified: The audio track is perfectly synced with the speaker's mouth movements. Check visually at multiple points.

[ ] Audio Levels Normalized: The dialogue's volume is consistent and clear. Apply normalization or compression as needed.

[ ] Music Bed Balanced: The background music level is appropriate. It lowers when the speaker is talking and rises during pauses.

[ ] Video Quality Checked: The exported clip resolution is correct (e.g., 1080x1920 for vertical). There are no visible compression artifacts.

Phase 5: Branding & Graphics
[ ] Logo Present & Correct: The client's logo is visible, correctly sized, and positioned according to brand guidelines.

[ ] Captions/Audio Sync: The burned-in captions are perfectly timed with the spoken words. Check the timing of the final export.

[ ] End Card Applied: The branded end card appears for the correct duration (3-5 seconds) at the end of the video.

[ ] Transitions Consistent: All transitions are smooth and match the style of the template.

Phase 6: Export & Delivery
[ ] Export Settings Verified: The correct export preset is selected (e.g., H.264, Match Source High Bitrate).

[ ] File Exported: The final reel has been exported and saved in the client's designated delivery folder.

[ ] File Named Correctly: The output file follows the agreed-upon naming convention.

[ ] Final Review: The exported file is watched one last time in its entirety to catch any errors before delivery.

Final Sign-off:
I certify that all steps in this checklist have been completed to the XLNZ standard of quality.
